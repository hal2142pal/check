# 3.2.S.1 General Information (Caffeine)

## 3.2.S.1.1 Nomenclature

| Category | Name |
| :--- | :--- |
| **International Nonproprietary Name (INN)** | Caffeine |
| **United States Adopted Name (USAN)** | Caffeine |
| **Chemical Name (IUPAC)** | 1,3,7-Trimethyl-3,7-dihydro-1H-purine-2,6-dione |
| **Chemical Name (Alternative)** | 1,3,7-Trimethylxanthine |
| **Chemical Abstracts Service (CAS) Registry Number** | 58-08-2 |

## 3.2.S.1.2 Structure

- **Molecular Formula:** C₈H₁₀N₄O₂

## 3.2.S.1.3 General Properties

| Property | Description |
| :--- | :--- |
| **Appearance** | White or almost white, crystalline powder or silky, white or almost white crystals. |
| **Physical State** | Solid |
| **Solubility (at 25°C)** | - **Water:** Sparingly soluble (~16 mg/mL)<br>- **Boiling Water:** Freely soluble<br>- **Ethanol (96%):** Slightly soluble<br>- **Ether:** Slightly soluble |
| **Melting Point** | 234°C to 239°C |
| **pKa** | Approximately 0.6 (very weak base; pKa of protonated form) |
| **Hygroscopicity** | Non-hygroscopic in crystalline form. |
| **Polymorphism** | Caffeine exhibits polymorphism. Form I (high temperature) and Form II (stable at room temperature) are known. |
